
Flow
----

- Load page
getLocations() -> Dropdown of locations is displayed
getCategories() -> Slides for all categories are displayed

- User chooses location
-> Chosen location is displayed

- User sets a slide for a category (repeated for each category)
searchArea() -> Best matching area for category is displayed
resultArea() -> Best matching area for all categories is displayed

- Optional: User marks checkbox for sharing result with Hyperspace ("Contact me")

- User submits result with own email in order to receive the result by email
